# 📺 ABTI 二次元灵魂测试 (Anime-Based Type Indicator)

> 突破次元壁，抛弃枯燥的心理学，用最纯正的动漫梗测出你隐藏的专属二次元人设！

这是一个基于全网爆火的“SBTI”衍生的纯前端互动小游戏。它通过 12 道涵盖经典动漫名场面的测试题，将玩家分配到 16 种截然不同的二次元人格模型中（如“高冷 B-King”、“不屈的打工魔王”、“经典败犬青梅”、“家里蹲吸血鬼真祖”等）。

## ✨ 核心特色

* **纯粹的二次元浓度**：题库包含泥头车转生、白学现场、天台中二宣言等经典刻在 DNA 里的梗。
* **硬核 4 维度计分系统**：完美复刻 MBTI 底层逻辑，分为四大对抗阵营：
    * ⚔️ **B** (Battle 热血战斗) vs 🍵 **D** (Daily 摸鱼日常)
    * 👁️ **C** (Chuunibyou 中二幻想) vs 📱 **T** (Tsukkomi 吐槽现实)
    * 😳 **A** (Tsundere 傲娇掩饰) vs ✨ **S** (Straight 直球坦率)
    * 😎 **O** (OP 龙傲天无敌) vs 😭 **M** (Mob 吃瘪路人)
* **极简架构，即开即用**：采用 React (CDN) + Tailwind CSS 构建，无需任何后端或环境配置（0 依赖，免 `npm install`）。
* **自适应 UI 设计**：完美适配手机端、平板与电脑端网页浏览，方便在社交圈转发。

## 🛠️ 部署与使用指南

### 本地运行
直接下载仓库中的 `index.html` 文件，双击使用任意现代浏览器（Chrome, Edge, Safari 等）打开即可流畅体验。

### 零成本快速上线 (通过 GitHub Pages)
如果你想生成一个链接发给朋友们玩，只需 3 步：
1. Fork 本项目，或新建一个公开仓库并将 `index.html` 上传。
2. 在你的 GitHub 仓库页面，点击右上角的 `Settings` -> 左侧边栏的 `Pages`。
3. 在 `Build and deployment` 模块中，将 `Branch` 选为 `main` (或 `master`)，点击 `Save`。
4. 等待 1-2 分钟刷新页面，GitHub 就会为你生成专属的在线访问链接！

## 💡 自定义与魔改

所有的题库、16 种人格结果和评分逻辑都在 `index.html` 的 `questions` 和 `resultsData` 数组中。你可以非常方便地：
- 替换成你更喜欢的动漫梗。
- 增加题目数量（只需确保各维度题目数量平衡）。
- 修改颜色、图标和二次元人设的文案。
